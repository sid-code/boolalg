Boolean algebra expression simplifier

Disclaimer: This simplifier uses an ad-hoc best-effort algorithm that runs in polynomial time. It is also very buggy. It will not simplify arbitrary boolean algebra expressions, but it can do most you'd encounter in an introductory undergraduate course ;)
